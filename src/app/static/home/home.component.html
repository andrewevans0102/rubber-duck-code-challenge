Welcome to the Ruber Duck Code Challenge!

What is this?

This is a code challenge aimed at making learning fun. Register yourself, and join the project Slack Org to keep up on updates. Each week record what you've learned. Each record has points. Every Friday scores will be tallied, and winners will be added to the scoreboard with emojis. Winners are encouraged to share a brief writeup of what they worked on in the Slack. Contact Andrew if you have any questions. Check out the docs to learn more. This project is Open Source on GitHub at the repo here.
